# Koda — Обратная связь

Этот репозиторий предназначен для сбора сообщений об ошибках и предложений по развитию от пользователей Koda.

* Официальный сайт: <https://kodacode.ru>
* Скачать Koda для JetBrains, VSCode или CLI: <https://download.kodacode.ru>
* Сообщество в Telegram: [@kodacommunity](https://t.me/kodacommunity)

## Правила участия

1. Перед созданием задачи проверьте, не существует ли уже [похожей задачи в общем списке](https://github.com/XCode-NLP/KodaCode/issues).
   Если существует -- не создавайте новую, просто проголосуйте за имеющуюся.
2. Используйте соответствующие шаблоны для создания задач.
3. Предоставляйте максимально подробную информацию.
   Это повысит вероятность, что мы сможем взять задачу в работу.
4. Будьте вежливы и конструктивны.

## Как сообщить об ошибке

Если вы обнаружили ошибку в работе Koda, пожалуйста, создайте задачу, используя шаблон [**«Сообщение об ошибке»**](https://github.com/XCode-NLP/KodaCode/issues/new?template=bug_report.yml).

Это поможет нам воспроизвести и исправить проблему.

**При сообщении об ошибке укажите:**
- Чёткое описание проблемы
- Шаги для воспроизведения
- Ожидаемое и фактическое поведение
- Скриншоты и логи, если применимо
- Ваше окружение (ОС, версия, браузер)

## Как предложить улучшение

Мы ценим ваши идеи по развитию Koda!
Создайте задачу, используя шаблон [**«Запрос новой функции»**](https://github.com/XCode-NLP/KodaCode/issues/new?template=feature_request.yml).

**В форме укажите:**
- Описание желаемой функции
- Проблему, которую она решит
- Альтернативные варианты (если есть)

## Отработка задач

Команда разработки отслеживает задачи, создаваемые в этом репозитории, проставляет корректные статусы и теги.

Если возможны обходные решения для ошибок, они будут предоставлены в этих задачах.
